DEVRENTAL - Car Rental Platform

Overview
DEVRENTAL is a comprehensive car rental management system built with Laravel, designed to simplify vehicle rental operations in Bandung, Indonesia. The platform offers an efficient solution for managing car rentals, customer transactions, and administrative tasks through a modern web interface.
